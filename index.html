WHY this project.

The reason why I build this project is that I want to share the detail of Human Face Detection with more programmer. Maybe, scientist don't care about the implementation but programmer DO! It's not perfect at this moment. So I need your help and make this detection system more and more powerful.

How To Use

Read all about it ! You can treat it like a tutorial about how face detection system works. After you know how it works, you can use your brilliant idea to modify the source code and rebuild this project into a more powerful one.

It's also a good opportunity to get more experience with Python, if you are beginner of Python

Don't forget to give a feedback with your pullrequest.

It's a pity that E-Face can't achieve the performance like OpenCV for human face detection. But we do the same great thing! This detection system are programmed in pure Python(2.7)
